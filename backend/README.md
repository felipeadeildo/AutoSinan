# AutoSinan

A bot collection to automatize some tasks on Sinan Online.

Avaliable bots:

- `InvestigationBot`
- `NotificationBot` (coming soon)

### Dependencies

- Python 3.10+
- Python Dependencies (`requirements.txt`)

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
